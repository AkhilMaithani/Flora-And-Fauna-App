# Flora and Fauna Android Application

<p>It is a static android application that shows flora and fauna present in Uttarakhand India.</p>

<p>The main purpose of this project is to build a mobile application that can be used and accessed by anyone. The technology working behind in such a way to give a look at Uttarakhandâ€™s Biodiversity. There is a lot more to know about the Biodiversity of Uttarakhand India. Here we are providing some keys to it so that one can easily go through some of the diverse range of flora and fauna habitats in Uttarakhand India.
